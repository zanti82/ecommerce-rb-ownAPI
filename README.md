# ecommerce-rb-ownAPI
A robust, end-to-end e-commerce solution featuring a custom-built REST API, a dynamic JavaScript frontend, and a MySQL relational database. This project demonstrates a complete CRUD implementation for product management, user carts, and order processing.

Key Features

    Custom API: Built from scratch to handle business logic, authentication, and data persistence.

    Dynamic Frontend: Responsive UI built from scartch that communicates asynchronously with the backend.

    Database Management: Optimized MySQL schema for products, users, and shopping carts.

    Persistent Cart: Secure "Add to Cart" functionality using database storage.

    Error Handling: Implementation of try-with-resources for safe database connection management.
